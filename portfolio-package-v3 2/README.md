## 🚀 Quick Start

### Option 1: View Locally
1. Extract this package to your computer
2. Navigate to `website/index.html`
3. Open in your browser (Chrome, Firefox, Safari, Edge)

### Option 2: Deploy to Web Server
1. Upload contents of `website/` folder to your hosting
2. Ensure `index.html` is in the root directory
3. Update file paths if needed (see UPDATE-GUIDE.md)

### Option 3: GitHub Pages
1. Create a new repository
2. Upload `website/` contents to repo
3. Enable GitHub Pages in settings
4. Your portfolio will be live at `yourusername.github.io`
